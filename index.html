Features

Everything you need to
run agents in production

From development to monitoring, AgentBox provides the complete infrastructure for your agentic workloads.

One-Command Deploy

Deploy your agent with a single agentbox deploy command. Automatic builds, packaging, and versioning out of the box.

Isolated Sandboxes

Each session runs in its own isolated sandbox environment. Secure, reproducible, and completely independent from other executions.

Claude SDK Native

First-class integration with the Claude Agent SDK. Build sophisticated agents with tool use, memory, and multi-step reasoning.

SDK & CLI

Full-featured TypeScript SDK for programmatic control. Powerful CLI for local development, deployment, and log streaming.

Real-Time Monitoring

Stream logs from running sessions in real-time. Track session status, errors, and outputs via the CLI, SDK, or web dashboard.

GitHub Integration

Connect your repo for automatic deployments on every push. Seamless CI/CD pipeline for your agent code.

How It Works

From code to production
in three steps

01

Develop

Write your agent code using the Claude Agent SDK. Access session inputs through environment variables. Bundle with esbuild for the sandbox environment.

main.ts 
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();
const input = process.env.AGENTBOX_SESSION_INPUT;
const sessionId = process.env.AGENTBOX_SESSION_ID;

const message = await client.messages.create({
  model: "claude-sonnet-4-5-20250929",
  max_tokens: 1024,
  messages: [{ role: "user", content: input }],
});

console.log(message.content);
02

Deploy

Deploy with a single CLI command or connect your GitHub repository for automatic deployments on every push. Each deployment is versioned and tracked.

terminal 
$ agentbox deploy
Building project...
Packaging artifact...
Uploading...
Deployed v3 → dep_x7k2m9p1
03

Execute & Monitor

Trigger agent sessions programmatically via the SDK. Pass inputs, secrets, and parameters. Monitor execution with real-time log streaming.

run.ts 
import { Agentbox } from "@agentboxsh/sdk";

const agentbox = new Agentbox({ apiKey: process.env.AGENTBOX_KEY });

const session = await agentbox.sessions.create(
  "proj_a1b2c3",
  { input: { url: "https://example.com", task: "analyze" } }
);

// Stream logs in real-time
const logs = await agentbox.sessions.logs(session.id);
for await (const log of logs) {
  console.log(log.message);
}
SDK Reference

Programmatic control
over every detail

Manage projects, deployments, sessions, and API keys programmatically with the TypeScript SDK.

Projects

Create, list, retrieve, and delete agent projects. Each project is a container for deployments and sessions.

agentbox.projects.create({ name })

Deployments

Upload artifacts with configuration specifying entrypoint and runtime. Node 22 and Python 3.12 supported.

agentbox.deployments.create(projectId, artifact)

Sessions

Create sessions with input data and secrets. Stream logs, filter by status, and terminate running sessions.

agentbox.sessions.create(projectId, { input })

API Keys

Generate, list, and revoke API keys for authentication. Keys are scoped to your account.

agentbox.apiKeys.create({ name })
